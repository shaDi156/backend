Backend Setup (Node.js)

Please follow the steps below to run the backend project correctly:

Download all attached project files.

Import the attached SQL file into phpMyAdmin to create the required database.

Ensure that Node.js is installed on your system.

Open the project directory in a terminal and run:

npm install


Start the backend server by running:

npm start


Once completed, the backend will be running and connected to the database.# backend
